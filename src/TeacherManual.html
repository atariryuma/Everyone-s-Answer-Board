🚀 はじめての授業 (3 分)

いちばん簡単な始め方は、🎒 新しい授業を作る ボタンからテンプレートを 1 つ選ぶことです。 フォームと回答先スプレッドシートが自動で作成され、すぐに使い始められます。

1

管理パネルを開く

学校から共有された URL にアクセスし、Google アカウントでログインしてください。 初回は「Google スプレッドシートへのアクセスを許可」と表示されるので「許可」を押します。

2

「🎒 新しい授業を作る」をクリック

画面上部の 🎒 レッスン セクションでボタンを押すと、授業フローのテンプレートが選べます:

授業の定番（3段階）・低学年向け・探究（田村モデル）・議論前後（2段階）

各フェーズには適切な可視化モード (数直線/マトリクス/円グラフ等) が自動で割り当てられます。 詳しくは 📊 可視化モード を参照。

3

3 ステップのウィザードで授業を作成

① 基本情報 (授業名・対象クラス) → ② フェーズ設計 (各フェーズの名前 / 形式 / 質問 / 1〜5 個) → ③ 確認 の 3 ステップ。

最後の確認画面で 「🎬 すべて作成して開始」 を押すと、各フェーズの Google フォームと回答先スプレッドシートが自動作成され、列マッピングが推定されて授業が開始されます。 そのまま画面下部の 「📣 公開する」 を押せば、児童が回答ボードを見られるようになります。

💡 単発のボードを使いたいときは： レッスン機能を使わず「フォームを連携」欄に Google フォームの URL を貼り付けるだけでも始められます。 新規 Form は「🔧 他の方法」を展開して「✨ テンプレートを作成」から シンプル掲示板 / 数直線 / マトリクス / 円グラフ の 4 種類のいずれかを選んで作れます。

📊 可視化モード

授業の目的に合わせて、回答の見せ方を選べます。テンプレートから作ると自動で適切なモードが設定されます。 既定の「自動判定」は、データの形 (線形尺度の有無等) を見て最適なモードを選びます。

📝
掲示板
回答を一覧表示。各カードに「いいね！」「なるほど！」「もっと知りたい！」のリアクションが付けられる。
👉 自由記述の意見交換に
🥧
円グラフ
選択肢の集計を円グラフで表示。多数派・少数派が一目で分かる。
👉 導入アンケート / 賛否を取る場面に
📊
ワードランキング
自由記述から頻出語を抽出し、頻度順に棒グラフで表示。
👉 振り返り / 学んだことの集約に
📏
数直線 (1 軸)
1〜5 の値で立場を可視化。同じ位置に複数人が集まる様子が見える。
👉 賛成/反対 など 1 つの軸の意見分布に
🎯
マトリクス (2 軸)
X・Y 2 軸の散布図。考えの背景や 2 つの価値の重み付けを比較できる。
👉 道徳 / 判断の根拠を可視化したい場面に
🔄
自動判定 (既定)
フォームに「線形尺度」列があれば数直線/散布図、無ければ掲示板に自動切替。
👉 とりあえず始めたいときに
モード切替：管理パネル ③「表示と公開」セクションの 📊 表示方法を細かく調整 を開くと、 モードを手動で変更できます。

🎙️ 議論支援ツール

児童画面の上部に出るツールバーで、議論の進行を支援できます。すべて教師が意図的に押したときだけ動くので、 児童に「いつ何を見せるか」は先生が制御できます。

📌

議論前

議論を始める前に押すと、その時点の分布をスナップショットとして保存します。 あとで「⇄ 比較」や「→ 動き」を使うときの基準点になります。

⇄

比較

議論前と現在を左右 2 画面で並べて見せます。 「クラス全体としてどう変わったか」を集団的に把握するのに向いています。

→

動き (マトリクス専用)

議論前 → 現在 の動きを軌跡 (細い線) で重ね描きします。1.5 秒のアニメーションで「動きが起きた」を直感的に伝えます。 クリックで再生、Shift+クリックで OFF。

💡 動きが小さい児童・動かなかった児童は軌跡が無いだけで、「考えを保持した」も等価な学びとして扱う設計です。 「大きく動いた = 正解」という見方にならないよう、強調色は使っていません。

⏱

軌跡スライダー

1 投稿 = 1 ステップ単位で過去の任意時点を再生できます。 「どの瞬間に意見が動いたか」を時系列で振り返るのに使います。「↺ 現在」ボタンで最新に戻ります。

使う順番の目安： 授業冒頭で 📌 議論前 を押す → 議論 → → 動き または ⇄ 比較 で振り返り。 プロファイル切替や軸ラベル変更で議論前スナップは自動でリセットされます (違うフォーム間の座標を結ばない安全弁)。

⚙️ 列設定とは？

スプレッドシートの「どの列」に「どんなデータ」が入っているかをアプリに教える設定です。 フォームから自動作成した場合は、ほぼすべて自動推定されます。

必須の列 (モードによって 1〜2 つ)

赤い 【必須】 ラベルが付いている列は必ず設定が必要です。ラベル自体もモードで変わります:

  • 掲示板: 💬 メイン質問列 (回答が入る列)
  • 円グラフ: 🥧 集計する選択肢列 (選択肢が入る列)
  • ワードランキング: 💭 自由記述列 (自由記述が入る列)
  • 数直線: 📏 X軸の数値列 (1〜5 等の整数列)
  • マトリクス: 📏 X軸 + 📐 Y軸 の 2 つの数値列

任意の列

📋 補助/理由列・🏫 クラス列・👤 名前列 は任意です。 設定するとホバー時の補助表示やクラス別フィルタが使えます。 不要なら空のままで OK。

🎯 自動推定とは: 列のヘッダー (質問文) と実データの形 (数値か文字か、長さ等) から、アプリが最適な役割を提案します。 各列の下に 🎯 自動推定 95% のようにバッジが表示され、信頼度が高ければそのまま使えます。 間違っていれば dropdown から手動で選び直してください (バッジは「✋ 手動設定」に変わります)。
🧭 軸ラベル自動取得: 数直線/マトリクスモードで使う X軸・Y軸の両端ラベル (例:「効率優先 ↔ 倫理優先」) は、 Google フォームの「線形尺度」項目に書かれているラベルから自動で取得されます。 空欄なら自動で埋まり、placeholder にもフォームの値が出るので、教師が手入力する手間はほぼゼロです。 自分の言葉に書き換えたいときは普通に上書き入力すれば、その値が優先されます。

📣 公開と URL 共有

すべての必須列が緑色 (🟢) になったら、画面下部の 📣 公開する を押します。 児童に共有する URL は 📋 URLコピー ボタンで取得できます。

児童への URL 共有方法

  • Google Classroom に投稿
  • 板書 / QR コード化して掲示 (QR コード生成は別ツールで)
  • 授業用スライドに貼り付け
⚠ ドメイン制限: 児童は学校ドメインの Google アカウントでログインする必要があります。 個人 Gmail からはアクセスできません (心理的安全性のため)。

⭐ 反応・ハイライト

児童は他の児童の回答に 👍 いいね！・💡 なるほど！・🔍 もっと知りたい！ の 3 種類のリアクションを付けられます。 先生は ⭐ ボタンで重要な回答をハイライトできます (児童画面では枠が光ります)。

初期設定はリアクション数 非表示: 心理的安全性のため、リアクション数は既定で児童には見えません。 授業の場面に応じて「リアクション数を表示」をオンにできます (管理パネル ③ セクション)。

🎒 レッスン (複数フェーズ授業)

1 つの授業を「導入 → 本時 → 振り返り」のように複数フェーズに分けて記録できます。 フェーズを切り替えると、児童画面の質問も自動で切り替わります。 授業終了後、各フェーズの回答が時系列で保存され、後日「振り返り画面」で再生できます。

1

授業を作成

「🎒 新しい授業を作る」→ テンプレート選択 → クラス・各フェーズの質問文を入力。

2

授業を実行

「🎬 すべて作成して開始」を押すと、フェーズ 1 のフォームが自動公開されます。 授業の進行に合わせて 「▶ 次のフェーズへ」 を押し、児童画面の質問を切り替えていきます。

3

授業を終了 → 振り返り

「⏹ 授業終了」 を押すと全フォームの受付が締め切られ、各フェーズの回答が時系列で保存されます。 レッスン一覧の 「振り返り」 ボタンから後日いつでも再生できます。

📋 プロファイル (複数ボード)

1 つのユーザーアカウントで複数の回答ボードを切り替えて使えます。 例: 「5年1組 国語」「5年1組 道徳」「6年2組 算数」を別々のプロファイルとして保存。

右パネル「📚 保存済みボード」の見方

現在編集中の行には水色の枠と次のバッジが表示されます:

  • 🟢 公開中 — このボードが児童に公開されている
  • 「現在」 — 編集中だが未公開

プロファイルの操作

  • 保存: 管理パネル下部の「💾 [プロファイル名] に保存」(現在のプロファイルに上書き)
  • 切替: 一覧の 📥 アイコン (非アクティブ行のみ)
  • 名前変更: 一覧の名前部分をダブルクリック
  • 削除: 一覧の 🗑 アイコン
  • 別名で保存: 「📋 別名で保存」を展開してプロファイル名を入力 → 新規保存
💡 入力は自動保存: 設定変更は約 1 秒後に自動保存されます。明示的な保存ボタンは不要です (プロファイルへの保存は別操作)。

🔒 プライバシー設定

既定では児童のプライバシーを最大限保護する設定になっています:

  • 👤 回答者名を表示: OFF (匿名表示)
  • 📊 リアクション数を表示: OFF
  • 🔒 ドメイン制限: 学校アカウントのみアクセス可
⚠ 公開後に名前を ON にした場合: 既に投稿された匿名の回答も実名で表示されます。事前に児童と合意した方針で運用してください。

🆘 困ったとき

フォーム URL を貼ったが「接続できません」と出る

Google フォームの編集用 URL (末尾が /edit) を使ってください。 回答用 URL (末尾が /viewform) は接続できません。

列設定で「必須: 0/2」のままで公開できない

選んだ表示モードが必要とする列が設定されていません。 例: マトリクスモードなら X軸・Y軸の数値列が両方必要。 ラベルの【必須】を確認してください。

児童が「公開されていません」と表示される

管理パネル下部の 📣 公開する を押し、ボタンが「🟢 公開中」に変わっていることを確認してください。 また、児童が学校ドメインの Google アカウントでログインしているか確認してください。

自動推定の信頼度が低い (50% 以下)

ヘッダー (質問文) に「回答」「理由」「クラス」などのキーワードが含まれていないと低く出ます。 低くても保存マッピングが優先されますが、手動で正しい列を選んでください。

授業を再公開したい (一度終了した後)

右パネル「保存済みボード」の対象プロファイルを 📥 で開き、「📣 公開する」を再度押してください。 公開状態は明示操作でのみ変わります。

意図しない列が dropdown に表示される

タイムスタンプ、UNDERSTAND/LIKE/CURIOUS/HIGHLIGHT などのシステム列は自動的に除外されます。 それ以外の余計な列が出る場合はスプレッドシートのヘッダーを確認してください。

1 ヶ月以上前の授業データを見たい

終了済みのレッスンは画面上部「🎒 レッスン」一覧の 「振り返り」 ボタンから時系列再生できます。 回答データはスプレッドシートに保存され続けるため、Drive から直接開くこともできます。

児童に見せる前に自分で表示を確認したい

公開すると画面下部に「現在の回答ボード」のフッターが現れ、「📊 ボードを開く」 ボタンで自分のブラウザで実画面を確認できます。 リアクションやハイライトも実際に操作して試せます。

スプレッドシートはどこに保存されている？

右パネル「📚 リソース」の 「📊 スプレッドシートを開く」 ボタンから直接開けます。 Google Drive の「マイドライブ」配下に「みんなの回答ボード」フォルダが自動作成され、フォーム本体と回答先スプレッドシートが一緒に整理されます。

複数クラスで同じフォームを使い分けたい

レッスン作成時に 対象クラス を複数選ぶと、1 つのフォームを全クラスで共有できます (児童が回答時にクラスを選択)。 クラス別に集計したいときは、ボード画面のクラスフィルタで切り替えられます。
