📚 みんなの回答ボード

完全操作マニュアル【先生向け】

生徒の回答をリアルタイムで共有・反応できる授業支援ツール

📋はじめに

ようこそ！「みんなの回答ボード」は、授業で生徒の意見や回答を集めて、クラス全体で見ることができるツールです。普段パソコンをあまり使わない先生でも簡単に使えるよう作られています。Googleフォームで生徒から回答を集めて、その内容をクラス全体で見たり、お互いに「いいね！」などの反応をつけたりして、みんなで意見を共有する楽しい授業ができます。

🚀簡単セットアップ

いつものGoogleアカウントでログインして、回答が入ったスプレッドシートをつなげるだけ。3分程度で使い始められます

👥リアルタイム共有

生徒が回答を送信すると、すぐに画面に表示されて、クラス全体でみんなの意見を見ることができます

⭐管理機能

特にいい回答に星マークをつけたり、不適切な回答を削除したりして、授業に合わせて画面を調整できます

💡このマニュアルの使い方

初めて使用される場合は「クイックスタート」から始めてください。詳細な設定方法は「詳細セットアップ」をご覧ください。日々の授業での使用方法は「日常操作」セクションで説明しています。左側の目次から必要な部分に直接ジャンプできます。

⚡3分クイックスタート

1

ログイン

学校から教えてもらったアドレス（URL）をクリックして、いつものGoogleアカウントでログイン

✓ 学校ドメインのアカウント推奨

2

シート接続

生徒の回答が保存されているGoogleスプレッドシートをつなげる

✓ アドレスをコピーして貼り付けるか、一覧から選ぶ

3

公開

「今すぐ公開」ボタンを押して生徒が見られるようにする

✓ 生徒に教えるアドレスは画面下部でコピーできます

💡 最も簡単な始め方

Googleフォームで質問を作成し、生徒に回答してもらってください。フォームから自動生成されるスプレッドシートをそのまま回答ボードに接続するのが一番簡単で確実な方法です。フォームのタイトルや質問文は自動で回答ボードのタイトルとして表示されます。

🔧詳細セットアップガイド

1

アプリへのログイン

学校から教えてもらった回答ボードのアドレス（URL）をクリックしてください。ページが開いたら、画面に表示される指示に従って、いつも使っているGoogleアカウントでログインしてください。

詳細な手順：
  1. インターネットで回答ボードのアドレスを開く
  2. 「ログイン」ボタンをクリック
  3. Googleアカウント選択画面で学校のアカウントを選ぶ
  4. 「スプレッドシートへのアクセスを許可」と表示されたら「許可」をクリック
  5. みんなの回答ボードの初回セットアップ画面が表示
⚠️ 重要な注意点

初めて利用する際には「Googleスプレッドシートへのアクセスを許可してください」等のメッセージが表示されることがありますが、これは回答ボードがスプレッドシートのデータを読み書きするために必要な権限です。必ず「許可」して先に進んでください。

2

回答用スプレッドシートの接続

生徒の回答が保存されているGoogleスプレッドシートをアプリにつなげます。セットアップ画面で、どのスプレッドシートを使うかを選ぶ画面が表示されます。

方法1: URLを直接入力
  1. 対象のスプレッドシートをブラウザで開く
  2. アドレスバーのURLを全選択してコピー
  3. ウィザードの入力欄に貼り付け
  4. 「🔍 URL検証」ボタンをクリック
  5. シート名とフォーム情報が表示されることを確認
方法2: 一覧から選択
  1. 「📋 一覧から選択」タブをクリック
  2. Googleドライブの一覧が表示されるまで待機
  3. 目的のスプレッドシートを選択
  4. 適切なシート名（タブ）を選択
  5. 関連フォームが自動検出されることを確認
💡 推奨設定

Googleフォームで作った質問の回答が自動で保存されたスプレッドシートを使うのがおすすめです。フォームを使うと、質問のタイトルなどが自動で読み取られて、設定がとても簡単になります。また、生徒も「📝 回答フォーム」ボタンから直接回答できるようになります。

3

回答列の設定（列マッピング）

スプレッドシートをつなげると、次はそのシートのどの列（縦の並び）に何のデータが入っているかを教える画面になります。これは回答ボードが正しく情報を表示するために大切な設定です。

設定する列の種類：
  • 💬 回答列：生徒の実際の回答文章が入っている列（必須）
  • 📋 理由列：回答の理由や説明が入っている列（あれば設定）
  • 🏫 クラス列：クラス名や学級が入っている列（あれば設定）
  • 👤 名前列：生徒の名前が入っている列（あれば設定）
設定手順：
  1. 各項目のプルダウンメニューを開く
  2. 適切な列名を選択
  3. 該当しない項目は「未選択」のまま
  4. 「🤖 AI列判定」で自動推測も可能
  5. 「設定を確認」で最終チェック
⚠️ 列設定の注意点
  • • 回答列は必須です。必ず生徒の主要な回答が入っている列を選択してください
  • • 理由列、クラス列、名前列は任意です。該当する情報がない場合は無理に選ぶ必要はありません
  • • AI列判定機能を使うと、列名から自動的に適切な列を推測してくれます
  • • 間違って設定した場合は、後から管理画面で変更できます
4

ボードの公開と表示設定

列の設定が完了したら、いよいよ回答ボードを公開します。公開前に、生徒にどのような情報を表示するかを設定できます。

表示オプション：
  • 👤 回答者名を表示：
    チェック時：生徒の名前が回答と一緒に表示
    （デフォルト：チェックなし＝匿名表示）
  • 📊 リアクション数を表示：
    チェック時：各回答に付いたリアクションの数を公開
    （デフォルト：チェックなし＝非表示）
公開手順：
  1. 表示オプションを確認・調整
  2. プレビューで表示内容を確認
  3. 「今すぐ公開」ボタンをクリック
  4. 画面下部に生徒用URLが表示されることを確認
  5. 「📋 URLコピー」で生徒用リンクを取得
💡 プライバシー配慮

初期設定では生徒のプライバシーに配慮して、名前とリアクション数は表示されません。授業の目的や生徒の年齢に応じて適切に設定してください。匿名表示により、生徒がより自由に意見を表現できる環境を作ることができます。

📊 設定完了後の画面イメージ

セットアップが完了すると、管理画面に「公開中」ステータスが表示され、画面下部に生徒用URLのコピーボタンが現れます。この状態で生徒は回答ボードを閲覧し、リアクションを付けることができるようになります。

📊日常操作ガイド

📋 回答ボードの見方

生徒用URLにアクセスすると、公開された「みんなの回答ボード」が表示されます。先生自身も生徒用URLから実際の見え方を確認できます。

画面構成の説明：
  • 上部：質問タイトル（フォームタイトルから自動取得）
  • 左側：現在の回答数、並び順設定（新着順/ランダム順/スコア順）
  • 右上：管理モード切替ボタン（歯車アイコン）
  • 中央：回答カード一覧
  • 下部：カードサイズ調整スライダー
操作パネルの機能：
  • 🏫 クラスフィルター：特定のクラスの回答のみ表示（設定時のみ表示）
  • 🔄 並び順：新着順・ランダム順・スコア順から選択
  • 📝 回答フォーム：生徒がここから回答入力（フォーム設定時）
  • ⚙️ 管理モード：先生専用の管理機能（歯車アイコン）
👁️ 先生と生徒の見え方の違い

先生がログインして閲覧している場合、画面右上に管理モード切替ボタン（⚙️歯車アイコン）が表示されますが、生徒には表示されません。また、管理モードに切り替えると、各回答カードに⭐ハイライトボタンや🗑️削除ボタンが表示されます。

💙 リアクション（反応）機能

生徒は他の人の回答を読んで、3種類のリアクションで反応を示すことができます。

👍いいね！

すてきだと思ったときに押すリアクション。サムアップアイコンで表示されます。

💡なるほど！

参考になったときに押すリアクション。電球アイコンで表示されます。

🔍もっと知りたい！

もっと知りたいときに押すリアクション。虫めがねプラスアイコンで表示されます。

リアクションのルール：
  • • 各回答に対して一人一回リアクションできます
  • • 同じボタンをもう一度押すとリアクションを取り消せます
  • • 複数の種類のリアクションを同時に付けることも可能です
  • • リアクション数の表示は先生の設定により変更できます

🔔 新着回答の通知

授業中に新しい回答が投稿されると、リアルタイムで通知されます。

通知の仕組み：
  1. 生徒がGoogleフォームで回答を送信
  2. 数秒後に画面上部に「📋 新しい意見が投稿されました」バナーが表示
  3. 「更新して表示」ボタンをクリック
  4. 最新の回答を含む一覧が表示される
  5. バナーは一定時間後に自動で消える
💡 授業での活用方法

新着通知を活用して、「また新しい意見が来ましたね」「今度はどんな考えでしょうか」などと生徒の関心を引きつけ、継続的な参加を促すことができます。通知が来たタイミングで更新し、新しい回答をクラス全体で確認する流れを作ると効果的です。

⚙️管理者機能（先生専用）

🔒 管理者権限について

以下の機能は先生（管理者）のみが使用できます。生徒には表示されません。管理機能を使用するには、回答ボード画面右上の⚙️歯車アイコンをクリックして管理モードに切り替えてください。

🔧 管理モードの切替

切替手順：
  1. 回答ボード画面の右上にある⚙️歯車アイコンを確認
  2. 歯車アイコンをクリックして管理モードに切替
  3. 各回答カードに管理ボタン（⭐削除ボタンなど）が表示されることを確認
  4. 管理作業完了後、再度歯車アイコンをクリックして通常モードに戻る

⭐ ハイライト機能

特にクラスで取り上げたい優れた回答やユニークな意見をハイライト（注目表示）としてマークできます。

ハイライトの付け方：
  1. 管理モードに切り替え
  2. 注目させたい回答カードを見つける
  3. カード上の⭐星形ボタンをクリック
  4. カードの背景色が変わることを確認
  5. 取り消す場合は再度⭐ボタンをクリック
効果的な活用方法：
  • • 模範回答や創意工夫のある回答をマーク
  • • 議論のきっかけとなる意見をハイライト
  • • 授業のまとめで取り上げたい回答を事前にマーク
  • • 多様な視点を示す回答を複数ハイライト
💡 授業での使い方例

「星が付いている回答に注目してください」「この金色に光っている回答、とても興味深い視点ですね」など、ハイライト機能を使って生徒の注意を特定の回答に向けることで、効果的な授業進行ができます。

🗑️ 回答削除機能

不適切な内容や個人情報が含まれる回答、授業に関係のない投稿などを削除できます。

⚠️ 削除機能の重要な注意事項
  • • 削除された回答は復元できません
  • • Googleスプレッドシートからも完全に削除されます
  • • 削除前に内容を十分確認してください
  • • 教育的配慮を持って慎重に使用してください
削除手順：
  1. 管理モードに切り替え
  2. 削除したい回答カードを確認
  3. カード上の🗑️ゴミ箱アイコンをクリック
  4. 確認ダイアログで「削除する」をクリック
  5. 回答がボードとスプレッドシートから削除される
削除すべき回答の例：
  • • 個人情報（本名、住所、電話番号など）が含まれる投稿
  • • 不適切な言葉や表現を含む回答
  • • 授業内容と全く関係のない投稿
  • • 他の生徒を傷つける可能性のある内容
  • • 明らかないたずらや意味のない投稿

👁️ 表示設定の変更

公開後も、生徒名やリアクション数の表示設定を変更できます。

設定変更手順：
  1. 管理画面（セットアップ画面）にアクセス
  2. 「表示設定」セクションを確認
  3. 「回答者名を表示」「リアクション数を表示」のチェックボックスを調整
  4. 「設定を更新」ボタンをクリック
  5. 回答ボードに反映されることを確認
💡 状況に応じた設定変更

授業の進行に応じて設定を変更できます。例えば、最初は匿名で自由な意見収集を行い、後半でより深い議論をする際に名前を表示するなど、段階的な公開設定が可能です。

👥生徒との共有方法

📋 生徒用URLの共有

ボードを公開したら、生徒がアクセスできるようにURLを共有します。

URL共有の手順：
  1. 回答ボード公開後、画面下部フッターを確認
  2. 「📋 URLコピー」ボタンをクリック
  3. クリップボードにURLがコピーされる
  4. メール、チャット、オンライン掲示板などに貼り付け
  5. 生徒に「このリンクから回答ボードを見てください」と案内
🔗 URLの特徴
  • 固定リンク：一度発行されたURLは変わりません
  • 再利用可能：次の授業でも同じURLを使用できます
  • ブックマーク推奨：生徒にブックマーク保存を推奨
  • 直接アクセス：QRコード化してプロジェクターに投影も可能

📝 回答フォームとの連携

Googleフォームを使用している場合、生徒は回答ボードから直接フォームにアクセスできます。

フォーム連携の流れ：
  1. 生徒が回答ボードURLにアクセス
  2. ページ左上の「📝 回答フォーム」ボタンを確認
  3. ボタンをクリックしてGoogleフォームを開く
  4. フォームで回答を入力・送信
  5. 数秒後に回答ボードに反映
生徒への説明例：

「みなさん、こちらのリンクにアクセスしてください」

「ページが開いたら、左上の『回答フォーム』ボタンを押してください」

「フォームに回答を入力して送信すると、みんなの画面に表示されます」

「送信後は、他の人の回答も見てリアクションしてみましょう」

🔐 生徒のアクセス権限

ログイン要件

生徒がボードを閲覧するには、Googleアカウントでのログインが必要です。学校発行のアカウント使用を推奨し、そのほうが安全で学校ドメイン内の利用であることが画面上にも表示されます。

🏫 学校ドメインアカウント
  • • ヘッダーに「学校ドメイン」と緑色で表示
  • • セキュリティが高く推奨
  • • 学校のポリシーに準拠
  • • 管理しやすい
🌐 外部アカウント
  • • ヘッダーに「外部アクセス」と表示
  • • 閲覧とリアクションは可能
  • • 可能な限り学校アカウント使用を推奨
  • • セキュリティ上の考慮が必要

🆘トラブルシューティング

❌ 回答ボードに何も表示されない
原因と解決方法：
  1. 公開状態の確認：管理画面で「公開中」になっているか確認し、未公開の場合は「今すぐ公開」をクリック
  2. データの確認：接続したスプレッドシートに実際に回答データが入っているか確認
  3. 列設定の確認：回答列が正しく設定されているか、列マッピング画面で再確認
  4. 権限の確認：スプレッドシートの共有設定で、アプリがアクセスできる権限があるか確認
🚫 生徒が回答ボードにアクセスできない
チェックポイント：
  1. URL確認：共有したURLが正確で、コピー時に文字が欠けていないか確認
  2. ログイン状況：生徒が適切なGoogleアカウントでログインしているか確認
  3. 学校アカウント：可能な限り学校発行のGoogleアカウントを使用するよう指導
  4. ブラウザ確認：古いブラウザや互換モードでないか確認、Chrome/Firefox/Safari等の最新版を推奨
  5. ネットワーク：学校のネットワーク制限により外部サイトへのアクセスが制限されていないか確認
📝 生徒が回答フォームにアクセスできない
解決手順：
  1. フォーム設定確認：Googleフォームの共有設定を確認し、「リンクを知っている全員」または「組織内の全員」に設定
  2. 権限設定：学校ドメイン限定の場合、生徒も同じドメインのアカウントでログインしている必要があります
  3. フォームステータス：Googleフォームが「回答を受け付けています」の状態になっているか確認
  4. 直接リンク配布：回答ボードのボタンが機能しない場合、フォームのURLを直接配布
🔄 リアクションが反映されない・遅い
対処方法：
  1. ページ更新：ブラウザの更新ボタンまたはF5キーでページを再読み込み
  2. ネットワーク確認：インターネット接続が安定しているか確認
  3. ブラウザキャッシュ：ブラウザのキャッシュとCookieをクリア
  4. 時間をおいて再試行：システムの負荷により遅延する場合があるため、少し時間をおいて再試行
  5. 別ブラウザで確認：問題が継続する場合、別のブラウザで同じ操作を試行
📊 スプレッドシートのデータが乱れた
⚠️ 重要な注意事項

LIKE、UNDERSTAND、CURIOUS、HIGHLIGHTなどの列は、システムが自動で追加する重要な列です。これらを削除すると回答ボードが正常に動作しなくなります。

安全な整理方法：
  1. バックアップ作成：整理前にスプレッドシートのコピーを作成
  2. システム列の保護：LIKE、UNDERSTAND、CURIOUS、HIGHLIGHTなどの列は削除・移動禁止
  3. データ追加：新しい列の追加は問題ありません
  4. 行の整理：回答行の削除は可能ですが、ヘッダー行（1行目）は保護
  5. 復旧方法：システム列を削除してしまった場合、管理画面から列設定を再実行
🔧 設定を変更したい
設定変更の手順：
  1. 管理画面アクセス：回答ボードの管理画面（セットアップ画面）にアクセス
  2. 接続変更：別のスプレッドシートに変更したい場合、新しいURLで再接続
  3. 列設定変更：列マッピングを変更したい場合、列設定画面で再設定
  4. 表示設定変更：名前表示やリアクション数の表示は表示設定で変更可能
  5. 設定保存：変更後は必ず「設定を更新」または「今すぐ公開」をクリック
❓ その他の問題・質問
サポート情報：
  • ログアウト：管理画面右上の「ログアウト」ボタンからサインアウト可能
  • システム管理者への連絡：技術的な不具合がある場合は、学校のシステム管理者にお問い合わせください
  • ブラウザ推奨環境：Chrome、Firefox、Safari、Edgeの最新版を推奨
  • セッション管理：長時間使用しない場合、自動でログアウトされることがあります

🔄 セッション管理のベストプラクティス

授業終了時の手順：
  1. 回答ボード上部の「公開終了」ボタンをクリック
  2. ボードが「未公開状態」になることを確認
  3. 生徒には「現在は準備中です」と表示される
  4. 次回使用時は同じURLで新しい内容を公開可能
💡 継続使用のメリット

生徒用URLは固定なので、一度教えれば毎回同じリンクを使用できます。生徒にブックマーク保存を推奨し、「いつもの回答ボード」として定着させることで、スムーズな授業運営が可能になります。
